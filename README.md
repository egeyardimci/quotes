# Env Values to Be Set

You need create and set the values in .env file in order the app to perform as intended.

## Values to be set:

### ACCESS_TOKEN_SECRET=

### REFRESH_TOKEN_SECRET=

### PORT=

### DB_URI=

# Starting the app

Run this commands:
`npm install`
`node app.js`
